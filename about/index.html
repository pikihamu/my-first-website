# my-first-website GitHub Pages で公開しているシンプルな静的サイトです。 - 公開URL: https://pikihamu.github.io/my-first-website/ ## 概要 HTML/CSS/JavaScript を学習するためのテストサイトです。トップページは `index.html` です。 ## ファイル構成（例） - `index.html` … トップページ - `css/` … スタイルシート - `images/` … 画像 - `js/` … スクリプト ## 更新方法 1. GitHub の Code 画面でファイルを編集し「Commit changes」します。 2. 1〜3分ほどで GitHub Pages に自動反映されます。 3. 反映が見えない場合はブラウザのハードリロード（Shift+再読み込み）を試してください。 4. 参照パスは相対パスを推奨（例: `css/style.css`, `images/logo.png`）。 ## ライセンス (c) 2025 pikihamu.
